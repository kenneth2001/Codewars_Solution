# Even or Odd

## Description 

Create a function (or write a script in Shell) that takes an integer as an argument and returns "Even" for even numbers or "Odd" for odd numbers.

## Test Cases

    2 -> "Even"
    0 -> "Even"
    7 -> "Odd"
    1 -> "Odd"
    -1 -> "Odd"
