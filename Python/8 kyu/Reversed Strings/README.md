# Reversed Strings

## Description 

Complete the solution so that it reverses the string passed into it.

Examples:

    'world' => 'dlrow'

## Test Cases

    'world' -> 'dlrow'
    'hello' -> olleh'
    '' -> ''
    'h' -> 'h'
