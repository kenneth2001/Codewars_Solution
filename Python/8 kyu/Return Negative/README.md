# Return Negative

## Description 

In this simple assignment you are given a number and have to make it negative. But maybe the number is already negative?

Examples:

    make_negative(1);  # return -1
    make_negative(-5); # return -5
    make_negative(0);  # return 0

## Test Cases

    42 -> -42
    -9 -> -9
    0 -> 0
