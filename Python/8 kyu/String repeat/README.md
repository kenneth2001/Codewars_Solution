# String repeat

## Description 

Write a function called repeat_str which repeats the given string src exactly count times.

Examples:

    repeatStr(6, "I") // "IIIIII"
    
    repeatStr(5, "Hello") // "HelloHelloHelloHelloHello"

## Test Cases

    (4, 'a') -> 'aaaa'
    (3, 'hello ') -> 'hello hello hello '
    (2, 'abc') -> 'abcabc'
